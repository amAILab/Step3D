# Step3D

Минималистичный сайт ООО «СТЕП_3Д»: 3D-печать, промышленный дизайн, реверсивный инжиниринг, прототипирование, образовательные и сценические проекты.

## Ссылки

- Сайт: https://amailab.github.io/Step3D/
- Репозиторий: https://github.com/amAILab/Step3D
- Telegram-канал: https://t.me/STEP_3D_Lab
- Менеджер: https://t.me/step_3d_mngr

## Что внутри

- минималистичный лендинг в светлом стиле;
- анимированный SVG-захватчик Step3D;
- реальные кейсы из канала X.Lab и публичный инженерный кейс Kawasaki / UMATEX;
- stories-галереи по клику на карточки кейсов;
- WebP-оптимизация изображений с JPG fallback;
- fallback-форма заявки через email, если Telegram недоступен;
- OpenGraph/social preview;
- `robots.txt`, `sitemap.xml`, `404.html`;
- мобильная плавающая CTA-кнопка.
