# Session 02 — финальная шлифовка первого экрана App

Цель: довести `/app/` до минималистичного пользовательского входа в стиле Ollama: один экран, один путь, минимум действий, никакой витрины возможностей.

## Что изменено

- Первый экран стал спокойнее: `услуга → описание → номер S3D → файлы/статус`.
- Убраны декоративные/дублирующие элементы из launcher:
  - верхняя фейковая панель внутри телефона;
  - кнопка `Go`;
  - dock-иконки;
  - лишняя ссылка на gallery из hero;
  - лишние file-action кнопки до создания проекта;
  - дубль кнопки account.
- Количество `.btn` в `/app/` снижено до 4:
  1. `Создать заявку`
  2. `Получить номер проекта`
  3. `Дослать файлы`
  4. `Открыть статус`
- Проектная карточка стала проще: номер и следующий шаг вместо “кабинетной” логики.
- Блок “Что будет в ответе” сокращён визуально: только первые два ключевых смысла на UI, остальное не давит на первый путь.
